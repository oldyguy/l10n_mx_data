	Archivo para facilitar la comprensión del esquema cfdv33.xsd del SAT
		s[#]:	Campo alfanumérico con # posiciones
		s[]:	Campo alfanumérico sin # de posiciones especificado
		e[#]:	Campo numérico entero con # posiciones
		f[#.%]: Campo numérico flotante con # posiciones y % decimales
		f[]:	Campo numérico flotante sin # de posiciones especificado
		d[#]:	Campo de tipo fecha con # posiciones
		u[#]:	Campo de tipo UUID con # posiciones
		v[3.3]:	Campo numérico flotante con valor 3.3 (versión del CFDI)
		{}:		Grupo de datos repetibles

	En el archivo tdCFDI.xsd se definen todos los tipos de campos utilizados
	en el archivo cfdv33.xsd
	
	El archivo catCFDI.xsd contiene los catálogos utilizados para la
	generación de CFDIs.
